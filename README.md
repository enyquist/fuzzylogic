## Installation and Setup

Assumes you have Python3.10 and python3.10-venv installed.

Clone the repository:

```sh
git clone https://github.com/enyquist/fuzzylogic.git
```
Navigate to the directory:

```sh
cd fuzzylogic
```

Create venv:

```sh
make init
```
